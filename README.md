# 🌍 Uganda Climate Guard 2026

An AI-powered weather and disaster preparedness assistant for Uganda.

## 🚀 Features

* 🌦️ 7-day weather forecast (°C + Kelvin)
* 🚨 Landslide, Flood, and Drought risk detection
* 📊 Risk Score system (0–100%)
* 🟢🟡🔴 Alert system
* 🌱 Farmer advisory (MAM season)
* 📍 District-based analysis

## 🧠 How It Works

This app uses structured AI prompting to:

* Analyze district input
* Identify ecological zones
* Predict risks
* Provide emergency protocols
